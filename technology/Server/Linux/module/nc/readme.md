# NC 命令

## 一、安装


``` sh
1. 下载地址
  http://sourceforge.net/projects/netcat/files/netcat/
  下载 netcat-x.x.x.tar.gz 包

2. 安装
  ./configure -help --prefix=/path/netcat
  make
  make install

3. 测试
  /path/netcat/bin/nc -help

```
